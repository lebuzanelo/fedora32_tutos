Tomcat 9

Criado sábado 06 junho 2020

Para instalar o Tomcat 9, primeiro você precisa garantir que o wget esteja instalado no Fedora 32:

sudo dnf install wget

Feito isso, rode os seguintes comandos:

cd $HOME
wget http://www-us.apache.org/dist/tomcat/tomcat-9/v9.0.35/bin/apache-tomcat-9.0.35.tar.gz

Observação: Para este tutorial foi utilizada a última versão do Tomcat, 9.0.35. Quando for realizar a configuração, sugerimos acessar o site http://tomcat.apache.org/ para verificar a última versão e com isso copiar a url da versão atualizada para inserir no comando wget.

Após terminar a operação acima, descompacte o arquivo baixado com o seguinte comando:

tar -xvzf apache-tomcat-9.0.35.tar.gz

Agora renomeie a pasta para tornar o processo mais fácil:

mv apache-tomcat-9.0.35 tomcat9

Agora vamos configurar as variáveis, primeiro exporte a variável CATALINA_HOME:

echo "export CATALINA_HOME="~/tomcat9"" >> ~/.bashrc

Depois atualize o bash:

source ~/.bashrc

Agora configure o usuário administrador:

nano ~/tomcat9/conf/tomcat-users.xml

Adiciona as seguintes linhas antes da opção </tomcat-user>, altere o nome do usuário admin (opcional) e configure uma senha:

<role rolename="admin-gui" />
<user username="admin" password="Informe_uma_Senha" roles="manager-gui, admin-gui" />

Inicie o Tomcat com o seguinte comando:

cd ~/tomcat9
./bin/startup.sh

Desligue o Tomcat com o seguinte comando:

cd ~/tomcat9
./bin/shutdown.sh
